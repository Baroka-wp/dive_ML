# Écriture & compréhension de papiers scientifiques

_On lit des extraits d'articles (SPPNet, R-CNN, etc.) puis on vulgarise l'idée principale avec nos mots._

## 🎯 Objectifs
- Identifier les concepts clés d'un papier (problème, méthode, résultat).
- Rédiger une synthèse structurée (contexte → idée → limites).
- **Prérequis bibliothèques** : Python pour la prise de notes, outils de lecture PDF (aucune lib spécifique).

## 🪜 Séquence d'apprentissage progressive
1. Repérer dans le notebook les extraits fournis et surligner les mots-clés.
2. Remplir un tableau 'méthode / points forts / points faibles'.
3. Transformer ces notes en paragraphe clair destiné à un camarade de classe.

## 🧩 Snippets à compléter
```markdown
## Fiche de lecture
- **Problème posé** : TODO résumer en une phrase
- **Idée principale** : TODO expliquer avec un dessin ou une métaphore
- **Ce que j'ai trouvé surprenant** : ...
```

## 🔁 Variantes / exercices
- Comparer deux méthodes (ex: R-CNN vs YOLO) dans un tableau argumenté.
- Enregistrer une capsule audio où tu expliques le papier en 2 minutes.
- Imaginer une application concrète pour ton lycée et décrire le prototype.
